Kalau kamu sudah tertarik memasuki dunia SEO. Kamu pasti pernah bertemu meta-description.

Dari [situs Moz](https://moz.com/learn/seo/meta-description) **jumlah karakter maksimum untuk meta description adalah 155 - 160 karakter**.

## Apa yang terjadi jika lebih ?

Jika jumlah karakternya lebih, maka situs mesin pencari seperti Google akan memotongnya (truncate) deskripsi kamu. Jadi jangan sampai bagian penting dari meta-description tersebut melewat batas karakter yang dianjurkan.

## Bagaimana dengan jumlah minimal karakter ?

Pada dasarnya tidak ada aturan. Namun rekomendasi saya pribadi mulai paling tidak berada di 50 karakter.  Karena meta description bisa membantu orang lain melihat rangkuman dari konten kamu, sehingga ia bisa menentukan apakah link kamu yang mereka cari atau bukan.

## Apa itu meta description pada tag HTML ?

Meta description adalah elemen HTML yang menyediakan ringkasan singkat tentang konten suatu halaman web. Meskipun Google dan mesin pencari lainnya tidak menggunakan meta description secara langsung untuk meranking situs dalam hasil pencarian, elemen ini tetap penting karena berfungsi sebagai teks iklan dalam hasil pencarian.

Pengguna biasanya membaca meta description untuk mengetahui ringkasan tentang halaman web sebelum memutuskan untuk mengklik atau tidak. Oleh karena itu, penulisan meta description yang baik sangat penting untuk menarik perhatian dan minat pengguna, serta meningkatkan tingkat klik (click-through rate, CTR) dalam hasil pencarian.

Sebagai contoh, berikut adalah format dasar dari sebuah meta description:

`<meta name="description" content="Ini adalah ringkasan singkat tentang halaman web saya.">`

Meta description harus jujur dan akurat menggambarkan konten halaman, dan sebaiknya berisi kata kunci yang relevan dengan konten tersebut. Biasanya, panjang optimal meta description adalah antara 50-160 karakter.
